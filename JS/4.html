Condicionales

¿Qué son las condicionales?

Las estructuras condicionales permiten ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

La estructura if

let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
}

Con else y else if

let hora = 14;
if (hora < 12) {
  console.log("Buenos días");
} else if (hora < 18) {
  console.log("Buenas tardes");
} else {
  console.log("Buenas noches");
}

Operador ternario

Una forma corta de escribir una condición:

let acceso = edad >= 18 ? "Permitido" : "Denegado";
